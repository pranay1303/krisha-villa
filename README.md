```bash
npm run dev
```

## ⚠️ Important (node_modules)

**Do NOT push `node_modules` to GitHub**

Before pushing your code, make sure you have a `.gitignore` file with:

```bash
node_modules/
.next/
dist/
.env
```

👉 If `node_modules` is already pushed, remove it using:

```bash
git rm -r --cached node_modules
git commit -m "Removed node_modules"
git push
```
